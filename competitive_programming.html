Competitive Programming

Your roadmap to Codeforces Grandmaster: mindset, practice, and resources.

Mindset & Motivation

Achieving Grandmaster requires resilience, curiosity, and consistent effort. Embrace failures as learning opportunities, set daily goals, and track progress in a journal.

Structured Learning Path

  • Fundamentals: Data structures (arrays, trees, graphs), algorithms (sorting, searching).
  • Intermediate: Greedy, DP, DSU, segment trees, two pointers.
  • Advanced: Flows, FFT, advanced geometry, string algorithms.
  • Expert: Custom data structures, game theory, probabilistic methods.
